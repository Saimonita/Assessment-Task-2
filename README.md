## 7. Accessibility Audit (WAI Standards)
To ensure this application is inclusive and meets Web Accessibility Initiative (WAI) standards, a Lighthouse audit was conducted using browser Developer Tools.
* **Color Contrast:** Verified that foreground and background colors provide sufficient contrast for visually impaired users.
* **Screen Readers:** Verified that the site structure supports screen reading software (e.g., using Alt text).
